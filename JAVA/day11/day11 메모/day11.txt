[ (접근)제어자 ]

public (자바는 public이 디폴트 초기값)
-> 단순 연산자로 (s1.name = "대환" 처럼)
(외부로부터)의 접근을 막고 싶다 !

private
-> 해당 클래스 내부에서만 접근 가능

앞으로 "모든 멤버 변수는 private를 설정" 한다.

게터와 세터를 함께 생성 해 줘야 한다.

자동으로 만드는 기능이 있다.

외부에서 직접적으로 멤버변수의 값에 접근할 수는 없으나,
getter와 setter를 인지하고 있으면 값을 바꿀 수 있다 !

그래서 접근제어자는 "캡슐화" 와 많이 관련 되어 있다.
