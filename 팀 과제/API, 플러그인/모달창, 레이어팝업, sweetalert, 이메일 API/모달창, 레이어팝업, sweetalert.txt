1. 모달창 :
사용자의 이목을 집중시키기 위한 그래픽 인터페이스 창.

특정 기능을 수행할 때만, 부모 페이지 (웹 브라우저) 위에 나타난다.

부모 페이지 위에 나오기 때문에, 부모 페이지를 벗어나면,
해당 창도 사라진다.

부모 페이지를 조작할 수 없다.

2. 레이어 팝업 :
모달창과 비슷한 방식으로 동작 한다.

레이어 팝업으로 특정 사이트 위에 나타난다.

모달창과 목적은 비슷하지만, 차이점이 있긴 하다.

부모 페이지를 조작할 수 있다는게 모달창과의 차이점 이다.

모달창은 부모페이지를 조작할 수 없다.

3. sweet alert :
사용자에게 동적이고 시각적으로 매력적인 알림창을 제공하는
JavaScript 라이브러리 이다.

정보 메시지를 표현 할 수 있다.

사용자 정의 버튼 및 이벤트 처리 기능의 목적으로
표현 할 수 있다.
(확인 / 취소)
