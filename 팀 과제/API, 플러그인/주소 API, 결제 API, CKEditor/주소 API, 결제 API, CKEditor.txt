주소 API

1) 주소 검색이 가능해 진다.

2) 좌표 변환 기능도 제공한다.
=> 지오 코딩
(주소를 검색하면, 좌표로 자동으로 변환 해주는 기능)
(반대도 가능)
(좌표로 검색해도, 주소로 변환)

지도 API랑 연동해서 사용도 가능 하다.

애초에 좌표로 검색하려면, 지도가 있어야 하는..

3) 우편 번호도 제공 한다.

4) 유효성 검사 기능을 제공 한다.
사용자가 입력한 키워드 (단어)가
유효하지 않으면,
검색 결과가 없다고 뜨게 된다.

=> 결제했을 때, 오배송 되거나
하는 경우에 수를 방지 하게 된다.

----------------------

결제 API

토스 API를 사용 했다.

1) 간편 결제.

2) 다양한 결제 서비스를 지원 한다.
계좌이체, 쿠페이 머니, 신용/체크 카드, 법인카드,
휴대폰, 무통장입금, ..

3) 
결제 상태 확인 기능을 제공 한다.
언제 결제했는지, 등등..

결제 API는 이미 많이 보편화 되어 있다.

----------------------

CKEditor 플러그인

웹 애플리케이션 또는 웹 브라우저에서,
사용자가 쉽게 텍스트를 편집 할 수 있게
도와 준다.

글의 서식을 지정하거나,
이미지, 미디어 삽입 등,
표로 작성, 링크 삽입 등,
여러가지 다양한 편집 기능을
쉽게 할 수 있도록 도와 준다.

WYSIWYG(위지위그)
What you see is what you get

보는 대로 얻는다.

(사용자가 HTML 태그를 몰라도
간편하게 사용할 수 있는 플러그인
이라고 생각하면 될듯)

다양한 기기와 호환성이 맞아서,
어디 어디에서나 사용자가 편하게 
텍스트와 같은것을 쉽게 수정할 수 있도록
도와 준다.

오픈소스이고, 계속 개발이되고 업뎃이 된다.

편집 환경도 사용자가 입맛대로 구성해서,
사용 할 수도 있다.
