SNS 로그인 API

공통적인 5가지 단계

1) 각 sns 플랫폼에서 계정 생성
2) application 등록
3) 클라이언트 ID 와 비밀번호 발급 받기
4) 스크립트 불러오기와 로그인 버튼 추가 하기
5) 로그인 요청을 처리 하기

----------------------

콜백 함수 ~~

----------------------

아코디언 :
일반적으로 모든 섹션이 축소되어있어서
페이지 공간을 절약할수있다.
누르면 내부 내용이 열리고, 다시 누르면 축소된다.
(드롭다운)

탭스:
웹페이지를 간단히 요약할 수 있고,
링크를 걸어서 다른 페이지로 이동하게 할 수 있다.
사용자가 탭을 클릭하면 
일반적으로 버튼, 링크, html 요소 등으로 구성된다.

슬릭 슬라이더 :
"스와이퍼 행동을 기본적으로 제공"하고,
옵션을 스와이퍼 플러그인보다는 기능이 적지만,
그래도 쉽게 사용할 수 있다는 장점이 있다.
슬라이더를 동적으로 추가 및 제거
필터링으로 조건을 받아서 슬라이더별로
숨기고 나타내고 할 수 있다.

스와이퍼 플러그인 :
스와이퍼 란 ?
터치스크린의 손을 댄 상태로, 무언가 행동을 하는것을 의미한다.
모바일에서 자주 사용 한다.
다양한 자료를 사용하기에, 커스텀 해서 사용하기 좋다.
